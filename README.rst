Installation
================================================================================

You can install it via pip:

.. code-block:: bash

    $ pip install robotchef_britishcuisine


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/chfw/robotchef_britishcuisine.git
    $ cd robotchef_britishcuisine
    $ python setup.py install
